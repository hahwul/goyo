마크다운 문법

이 가이드는 Goyo 테마에서 렌더링되는 일반적인 마크다운 요소의 예시와 효과적인 글쓰기 팁을 제공합니다.

제목 (Headings)

콘텐츠를 논리적으로 구조화하기 위해 <h2>부터 <h6>까지 사용합니다.

## 이것은 H2 입니다
### 이것은 H3 입니다
#### 이것은 H4 입니다
##### 이것은 H5 입니다
###### 이것은 H6 입니다

강조

텍스트를 굵게 또는 기울임꼴로 만들 수 있습니다.

*기울임꼴 텍스트* 또는 _기울임꼴 텍스트_
**굵은 텍스트** 또는 __굵은 텍스트__
***굵고 기울임꼴 텍스트*** 또는 ___굵고 기울임꼴 텍스트___
~~취소선 텍스트~~

목록

순서 없는 목록

별표(*), 더하기 기호(+), 또는 하이픈(-)을 사용합니다.

* 항목 1
* 항목 2
  * 하위 항목 2.1
  * 하위 항목 2.2
- 다른 항목

렌더링 결과:

  • 항목 1
  • 항목 2
    • 하위 항목 2.1
    • 하위 항목 2.2
  • 다른 항목

순서 있는 목록

숫자와 마침표를 사용합니다.

1. 첫 번째 항목
2. 두 번째 항목
   1. 하위 항목 2.1
   2. 하위 항목 2.2
3. 세 번째 항목

렌더링 결과:

  1. 첫 번째 항목
  2. 두 번째 항목
    1. 하위 항목 2.1
    2. 하위 항목 2.2
  3. 세 번째 항목

할 일 목록 (GitHub 스타일 마크다운)

- [x] 완료된 작업
- [ ] 미완료 작업

렌더링 결과:

  • 완료된 작업
  • 미완료 작업

코드 블록 (Fenced)

세 개의 백틱(```)을 사용하고 구문 강조를 위해 언어를 지정합니다.

```python
def hello():
  print("안녕하세요, Goyo!")
```

```javascript
console.log("안녕하세요, Goyo!");
```

렌더링 결과:

def hello():
  print("안녕하세요, Goyo!")

console.log("안녕하세요, Goyo!");

가로줄 (Horizontal Rule)

세 개 이상의 하이픈, 별표, 또는 밑줄을 한 줄에 단독으로 사용합니다.

---
***
___

표 (Tables)

파이프(|)와 하이픈(-)을 사용하여 표를 만듭니다.

| 헤더 1   | 헤더 2   | 헤더 3   |
| :------- | :------: | -------: |
| 왼쪽 정렬 | 가운데 정렬 | 오른쪽 정렬 |
| 셀 2     | 셀 3     | 셀 4     |
| 셀 5     | 셀 6     | 셀 7     |

렌더링 결과:

헤더 1헤더 2헤더 3
왼쪽 정렬가운데 정렬오른쪽 정렬
셀 2셀 3셀 4
셀 5셀 6셀 7

각주 (Footnotes)

[^레이블]과 [^레이블]: 노트 내용을 사용하여 각주를 만듭니다.

이것은 각주가 있는 텍스트입니다.[^1]

그리고 또 다른 각주입니다.[^another]

[^1]: 이것은 첫 번째 각주입니다.
[^another]: 이것은 또 다른 각주입니다. **마크다운**도 포함할 수 있습니다.

렌더링 결과:

이것은 각주가 있는 텍스트입니다.1

그리고 또 다른 각주입니다.2
